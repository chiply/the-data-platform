## Codebase Patterns
- Pulumi config keys are namespaced by project name (e.g., `tdp-cluster:clusterType`), so stack config files must use the `config:` top-level key with fully qualified names.
- The provider dispatch pattern (switch on clusterType in index.ts, delegate to provider modules in `providers/`) is extensible.
- @pulumi/command is useful for wrapping CLI tools (like k3d) that don't have native Pulumi providers.
